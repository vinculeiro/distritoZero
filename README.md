# distritoZero

**distritoZero** é unha chispa que trata de prender a curiosidade. Como tal non recolle ideas por quen as emite senón polo boas que son.

👉 **[Visitar a web](https://vinculeiro.github.io/distritoZero/)**

---

## Que atoparás aquí

- **Análise de políticas** baseadas en evidencia e datos, non en opinións sen fundamento
- **Visualizacións e narrativas de datos** que axudan a entender problemas complexos de forma clara
- **Debates informados** sobre servizos públicos, gobernanza e calidade democrática
- **Propostas innovadoras** para mellorar a vida cotiá mediante tecnoloxía accesible, sustentabilidade e participación cidadá
- **Unha visión do territorio** como ecosistema de futuro, onde a comunidade e os datos se combinan para crear solucións novas
